Thunderbit

Una aplicacion web para la gestión de recursos digitales

Ver proyecto en GitHub

Etiquetas

Thunderbit organiza los recursos digitales mediante etiquetas, un método conocido y apliamente utilizado

Multiplataforma

Thunderbit puede ser ejecutado en distintas plataformas, como GNU/Linux, Windows y Mac
