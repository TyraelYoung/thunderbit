Thunderbit

Una aplicación web para la gestión de recursos digitales

Principales características

Etiquetas

Thunderbit organiza los recursos digitales mediante etiquetas, un método conocido y ampliamente utilizado

Multiplataforma

Thunderbit puede ser ejecutado en distintas plataformas, como GNU/Linux, Windows y Mac

Software Libre

El código fuente de Thunderbit es distribuido bajo la licencia libre GNU Affero General Public License
